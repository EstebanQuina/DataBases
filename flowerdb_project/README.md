# Flower Production Company Database 🌸

This repository contains the final project for implementing a relational database for a Flower Production Company. It includes the database schema and a Python-based application to perform CRUD (Create, Read, Update, Delete) operations.

## Project Structure
* `schema.sql`: Contains the MySQL commands to create the database, tables, and relationships.
* `python.py`: The main Python script that implements the CRUD operations to interact with the database.
* `requirements.txt`: Lists the Python dependencies required to run the application.

## Prerequisites
Before running this project, ensure you have the following installed:
* [MySQL](https://dev.mysql.com/downloads/)
* [Python 3.x](https://www.python.org/downloads/)

## Setup Instructions

**1. Database Setup:**
Open your MySQL environment (like MySQL Workbench or the command line) and run the `schema.sql` script to build the database structure.

**2. Install Python Dependencies:**
Open a terminal in the project directory and install the required packages:
